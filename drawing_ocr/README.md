

# Модуль распознавания штампов чертежей

## Назначение
Извлечение структурированной информации из основной надписи (штампа) чертежа с помощью vision‑language модели.

## Технологии
- **vLLM** (модель Qwen3‑VL) – быстрый инференс vision‑языковой модели.
- **Pydantic** – схема `StampInfo` для строгого выходного формата.

## Основные компоненты
- `StampInfo` – модель данных (материал, номер чертежа, название, дата, дополнительные поля).
- Агент `Agent` с `output_schema=StampInfo`, который анализирует изображение и возвращает заполненную схему.
- Промпт, заставляющий модель возвращать именно значения, а не названия полей.

## Пример использования
```python
response = agent.run(
    "Ты OCR‑агент. Считай текст из штампа чертежа...",
    images=[Image(filepath="ramka.jpg")],
    stream=True
)
print(response.content)

Выходные данные

Объект StampInfo, содержащий все распознанные поля штампа.
