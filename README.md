# Industrial Knowledge Base

**Локальная система обработки промышленной документации с использованием LLM**

## 🎯 Назначение
Автоматизированная система для:
- OCR документов и чертежей (MinerU + vLLM)
- NLP-обработки текста
- Суммаризации через Agno Agents
- Векторного поиска (ChromaDB)
- Безопасного доступа для различных категорий пользователей

## 📂 Структура
| Каталог | Описание |
|---------|----------|
| `Out_MD/` | Скрипт преобразования PDF в MD |
| `ocr_mineru/` | OCR документов через MinerU |
| `nlp_processing/` | Очистка и обработка текста |
| `drawing_extraction/` | Извлечение данных из чертежей |
| `vector_db_agents/` | Векторная БД и Agno агенты |
| `core/` | Конфигурация и безопасность |
| `docs/` | Обработанные документы |
