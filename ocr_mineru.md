# OCR модуль с использованием MinerU

## 📋 Описание
Модуль отвечает за распознавание текста из PDF-документов через сервис MinerU.

## 🔧 Основные компоненты

### Load_md_03.md
**Пайплайн обработки документов:**
- **Ячейка 1-2**: Импорты, конфигурация, загрузка spaCy
- **Ячейка 3-4**: Утилиты очистки текста и фильтрации ключевых слов
- **Ячейка 5-6**: Извлечение ключевых слов и именованных сущностей
- **Ячейка 7-8**: Клиент MinerU и извлечение текста (ТОЛЬКО PDF через MinerU)
- **Ячейка 9**: Чтение titul.md и объединение метаданных
- **Ячейка 10**: Улучшение MD-разметки (заголовки, списки)
- **Ячейка 11-12**: Обогащение метаданных и запуск Этапа 1
- **Ячейка 13-18**: Pydantic схемы, Agno Agent, суммаризация (Этап 2)

## 📤 Выходные каталоги
| Каталог | Формат | Содержимое |
|---------|--------|------------|
| `outDOC/` | .md | С YAML frontmatter (метаданные, keywords) |
| `outDOC_clean/` | .txt | Чистый текст для суммаризации |
| `outDOCmd/` | .md | Улучшенная разметка без метаданных |
